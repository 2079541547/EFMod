## Получение поля

```Cpp
EFModAPI::GetField<int /* Тип поля */>(
    {
        "A" /* Пространство имен */,
        "Class" /* Класс */,
        "API" /* Имя поля */
    }); // Получить поле, предоставляемое загрузчиком
```

## Вызов функции

```Cpp
EFModAPI::callFunction<int /* Тип возвращаемого значения функции */>(
    {
        "A" /* Пространство имен */,
        "Class" /* Класс */,
        "Func" /* Имя функции */,
        "int" /* Тип возвращаемого значения */,
        1 /* Количество параметров */
    },
    "Предположим, что здесь передается строка, но фактически это может быть любой тип" /* Параметр 1 */,
    ...
    ); // Вызвать функцию, предоставляемую загрузчиком
```
