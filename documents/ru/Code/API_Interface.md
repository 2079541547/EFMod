## Реализация интерфейса Mod

### Стандартный Mod

```Cpp
std::map<uint64_t, std::vector<void *>> get_hooks()
{
    /**
     * Выполните необходимые действия, например, регистрация хуков
     */
    EFMod::registerHook(
        {
            "A", /* Пространство имен */
            "Class", /* Класс */
            "Func", /* Имя функции */
            "int", /* Тип возвращаемого значения */
            1 /* Количество параметров */
        } /* Информация о методе для хука */,
        (void *)MyFun /* Логика, которую нужно вставить после хука; примечание: MyFun является функцией. Убедитесь, что вставляемая функция имеет те же параметры и используйте (void *) для преобразования типов, или это может привести к сбою */
    );                   // Хук метода Func в классе Class в пространстве имен A, который возвращает int и принимает 1 параметр
    return EFMod::hooks; // Возврат таблицы хуков
} // Эта реализация функции обязательна, даже если она возвращает пустую карту, ее приоритет между инициализацией API и до фактической инициализации
```
