## 获取字段

```Cpp
EFModAPI::GetField<int /* 字段类型 */>(
    {
        "A" /* 命名空间 */,
        "Class" /* 类 */,
        "API" /* 字段名称 */
    }); //获取加载器提供的字段
```

## 调用函数
```Cpp
EFModAPI::callFunction<int /* 函数类型 */>(
    {
        "A" /* 命名空间 */,
        "Class" /* 类 */,
        "Func" /* 函数名称 */ ,
        "int" /* 返回类型 */ ,
        1 /* 参数数量 */
    },
    "假设这里传入一个字符串，实际可以为任何类型" /* 参数1 */ ,
    ...
    ); //调用加载器提供的函数
```
